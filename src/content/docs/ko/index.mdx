<div class="hero-code">
    <h2 class="head">문제 해결, 모범 사례 학습</h2>

    **Biome**은 _JavaScript_, _TypeScript_, _JSX_, _CSS_ 및 *GraphQL*을 위한 [**고성능 린터**](https://github.com/biomejs/biome/tree/main/benchmark#linting)로, ESLint, TypeScript ESLint, 및 [기타 소스](/linter/rules-sources/)에서 [**<NumberOfRules/>개의 규칙**](/linter/rules/)을 제공합니다.

    **Biome**은 코드를 개선하고 더 나은 프로그래머가 되는 데 도움이 되는 **상세하고 문맥에 맞는 진단을 제공합니다!**

    <LinterExample />

    [플레이그라운드](/playground/) 또는 프로젝트에서 직접 Biome 린터를 사용해 보세요.

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome lint --write ./src
    ```

</div>


<div class="hero-code">
    <h2 class="head">모든 것을 한 번에</h2>

    코드를 개별적으로 포매팅하고 린팅할 수 있을 뿐만 아니라 **명령 한 번으로 모든 작업을 한 번에 수행할 수 있습니다!**

    모든 도구는 다른 도구와 원활하게 통합되어 웹 프로젝트를 위한 **일관된 도구 체인**을 만듭니다.

    `check` 명령으로 모든 도구를 실행하세요.

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome check --write ./src
    ```

</div>
