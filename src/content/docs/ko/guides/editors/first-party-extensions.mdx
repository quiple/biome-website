다음 확장은 Biome 팀과 [Biome 조직](https://github.com/biomejs)의 일부가 유지 관리하는 확장입니다.


### VS Code

Biome 에디터 통합을 통해 다음을 수행할 수 있습니다.

* 저장할 때 또는 _Format_ 명령을 실행할 때 파일을 포매팅합니다.
* 파일 린팅 및 코드 수정 적용

Visual Studio 마켓플레이스에서 공식 [Biome VS Code 확장](https://marketplace.visualstudio.com/items?itemName=biomejs.biome)을 설치하세요.

Biome을 기본 포매터로 설정하려면 [지원되는 파일](/ko/internals/language-support/)을 열고

1. *명령 팔레트*(보기 또는 <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)를 엽니다.
1. *문서 서식 프로그램*...을 선택합니다.
1. *기본 포매터 구성*...을 선택합니다.
1. *Biome*을 선택합니다.

[레퍼런스 페이지](/ko/reference/vscode)에서 자세한 내용을 확인하세요.


### IntelliJ

Biome IntelliJ 플러그인을 설치하려면 [공식 플러그인 페이지](https://plugins.jetbrains.com/plugin/22761-biome)로 이동하거나 다음 단계를 따르세요.

**JetBrains IDE에서:**

1. IntelliJ IDEA를 엽니다.
2. **설정**으로 이동합니다.
3. 왼쪽 메뉴에서 **플러그인**을 선택합니다.
4. **마켓플레이스** 탭을 클릭합니다.
5. ‘Biome’을 검색하고 **설치**를 클릭합니다.
6. IDE를 다시 시작하여 플러그인을 활성화합니다.

**디스크에서:**

1. Versions 탭에서 플러그인 .zip 파일을 다운로드합니다.
2. `⌘Сmd`를 눌러 IDE 설정을 연 다음 플러그인을 선택합니다.
3. 플러그인 페이지에서 설정 버튼을 클릭한 다음 *디스크에서 플러그인 설치*...를 클릭합니다.


### Zed

1. open the *Command Palette* (View or <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
2. select **zed: extensions**
3. search **Biome**
4. select **Install**

Read more in the [reference page](/reference/zed).
