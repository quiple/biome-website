---
title: 수동 설치
description: Biome을 수동으로 설치하세요.
---

아직 Node.js나 `npm`(또는 다른 패키지 관리자)을 사용하고 있지 않다면 Biome의 독립형 CLI 바이너리를 사용하는 것이 좋은 선택이 될 수 있습니다.
즉, Biome을 사용한다는 이유만으로 `package.json`을 사용할 필요는 없습니다.

:::note
이미 `npm` 또는 다른 패키지 관리자를 사용하고 있다면 패키지 관리자를 사용하는 것이 Biome을 [설치하는 가장 좋은 방법](/ko/guides/getting-started#installation)입니다.
이미 도구에 익숙할 것이고 설치 및 업데이트가 더 간단합니다.
:::


## 지원 플랫폼

Biome이 작동하려면 플랫폼에 맞는 바이너리를 선택해야 합니다. 다음 표가 도움이 될 것입니다.

| CPU 아키텍처 | Windows       | macOS                    | Linux         | Linux (musl)       |
| ------------ | ------------- | ------------------------ | ------------- | ------------------ |
| `arm64`      | `win32-arm64` | `darwin-arm64` (M1 이상) | `linux-arm64` | `linux-arm64-musl` |
| `x64`        | `win32-x64`   | `darwin-x64`             | `linux-x64`   | `linux-x64-musl`   |

:::note
Linux용 Windows 하위 시스템(WSL)에 Linux 변형을 사용하세요.
:::


## Homebrew

Biome은 macOS 및 Linux 사용자를 위해 [Homebrew formula](https://formulae.brew.sh/formula/biome)로 제공됩니다.

```shell
brew install biome
```


## 게시된 바이너리 사용

Biome을 설치하려면 GitHub의 [최신 CLI 릴리스](https://github.com/biomejs/biome/releases)에서 플랫폼에 맞는 실행 파일을 가져와서 실행 권한을 부여하세요.

```shell
# macOS arm (M1 이상)
curl -L https://github.com/biomejs/biome/releases/download/cli%2Fv<버전>/biome-darwin-arm64 -o biome
chmod +x biome

# Linux (x86_64)
curl -L https://github.com/biomejs/biome/releases/download/cli%2Fv<버전>/biome-linux-x64 -o biome
chmod +x biome

# Windows (x86_64, PowerShell)
Invoke-WebRequest -Uri "https://github.com/biomejs/biome/releases/download/cli%2Fv<버전>/biome-win32-x64.exe" -OutFile "biome.exe"
```

:::note
`<버전>`을 설치하려는 Biome 버전으로 바꾸어야 합니다.
:::

이제 `./biome`을 실행하는 것만으로 Biome을 사용할 수 있습니다.
